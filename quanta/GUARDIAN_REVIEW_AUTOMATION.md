## Automation Technical Specification

### Document Processing Pipeline

**Input**: New document enters system

**Process**:
1. **Parse content** for domain keywords
2. **Match to guardians** based on domain taxonomy
3. **Queue for review** in guardian-specific workflows
4. **Execute review stages** sequentially
5. **Track prime progression** (3, 5, 7, 11, 13, 17...)
6. **Trigger expansion/collapse** at designated points
7. **Integrate perspectives** into final document
8. **Publish to wellsprings** for guardian domains
9. **Monitor ongoing** for updates triggering re-review

**Output**: Multi-perspective enriched document + wellspring additions

### Guardian Assignment Logic

```python
def assign_guardians(document):
    guardians = []

    # Scan for entity types
    if contains_animal_content(document):
        animal_type = identify_animal(document)
        guardians.append(get_guardian_for(animal_type))

    # Scan for character classes
    if contains_character_class(document):
        class_type = identify_class(document)
        guardians.append(get_guardian_for(class_type))

    # Scan for perspectives
    if contains_perspective(document):
        perspective_type = identify_perspective(document)
        guardians.append(get_guardian_for(perspective_type))

    # Scan for concepts
    if contains_concept(document):
        concept_type = identify_concept(document)
        guardians.append(get_guardian_for(concept_type))

    return guardians
```

### Review Stage Triggers

```python
def process_document_through_guardians(document, guardians):
    # Stage 1: Identification (already done)

    # Stage 2: Primary reviews
    for guardian in guardians:
        guardian.review_primary(document)
        guardian.add_to_wellspring(document.filtered_content)

    # Stage 3: Secondary reviews
    related_guardians = find_related_guardians(guardians)
    for guardian in related_guardians:
        guardian.review_secondary(document)

    # Prime stage reviews
    for prime in [3, 5, 7, 11, 13, 17, 19, 23, 29...]:
        if document.review_count == prime:
            for guardian in guardians:
                guardian.review_at_prime(document, prime)

    # Expansion phase
    if document.stage == "expansion":
        for guardian in guardians:
            guardian.amplify_contribution(document)
            guardian.wellspring.grow_rapidly()

    # Collapse phase
    if document.stage == "collapse":
        for guardian in guardians:
            guardian.consolidate_perspective(document)
            guardian.wellspring.crystallize()

    # Final integration
    document.integrate_all_perspectives(guardians)

    # Publish to wellsprings
    for guardian in guardians:
        guardian.wellspring.receive(document.perspective_layers[guardian])

    # Monitor ongoing
    document.set_monitoring(guardians)
```
